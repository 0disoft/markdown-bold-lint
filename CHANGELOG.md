# Change Log

All notable changes to the "markdown-bold-lint" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

## [1.1.0] - 2026-01-17 KST

- Prewarm diagnostics for workspace Markdown files on activation.
- Debounce document change refreshes to reduce churn.
- Align the bold underline default with configuration.
- Add type-safe diagnostic codes.
- Add a lint sample Markdown file for manual testing.
- Open the test folder automatically when launching the extension host.

## [1.0.2] - 2026-01-11

- Lower the minimum VS Code engine to ^1.104.0 for Antigravity compatibility.

## [1.0.1] - 2026-01-04

- Initial release
